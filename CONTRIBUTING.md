## Getting Started

1. **Fork the repo**
2. **Clone your fork**
   ```bash
   git clone https://github.com/your-username/genai_wiki.git
   cd genai_wiki
   ```
3. **Create a branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Make your changes**
5. **Commit with a clear message**
   ```bash
   git commit -m "feat: add new prompt example for AKS documentation"
   ```
6. **Push and open a PR**
   ```bash
   git push origin feature/your-feature-name
   ```
